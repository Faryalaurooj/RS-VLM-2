# RS-VLM

RS-VLM is a hybrid vision–language object detection framework for robust
small object detection in remote sensing imagery.

## Features
- Multiscale CNN–Transformer backbone
- Oriented object detection
- CLIP-style vision–language alignment
- Zero-shot and few-shot detection
- Real-time inference

## Benchmarks
DOTA, xView, FAIR1M, NWPU VHR-10, UCAS-AOD, RSOD
